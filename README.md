# Financial Dashboard Projects - Power BI

Collection of financial analysis dashboards built in Power BI using public banking data.

## Purpose
Demonstrate financial analysis skills applied to:
- Asset and equity evolution
- Portfolio structure analysis
- Leverage and risk behavior
- Profitability (ROE and Net Income)
- Credit and investment composition

## Tools
- Power BI
- Power Query
- Excel

## Projects Included

### Itaú Financial Analysis
Financial dashboard analyzing the evolution of assets, equity, leverage and profitability using Itaú public financial statements (BRGAAP).

More projects will be added over time.
