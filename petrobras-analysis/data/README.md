# Data Source – Petrobras Financial Analysis

This folder contains the raw financial statement files used as the data source for the Petrobras Financial Dashboard.

---

## Content

The files include:

- Balance Sheet (Balanço Patrimonial)
- Financial statements in R$ (BRGAAP / IFRS)
- Quarterly reports used to build the dashboard

---

## Important Notes

- All values in the original files are reported in **R$ millions**
- Values were converted inside Power BI for proper financial scale representation
- These files are provided only for transparency and reproducibility of the analysis
- No modifications were made to the original financial data

---

## Usage

These files were used as the base for:

- Data transformation (Power Query)
- Financial modeling
- Dashboard creation in Power BI

---

## Disclaimer

All data belongs to Petrobras and is publicly available.  
This project is for **educational and analytical purposes only**.
